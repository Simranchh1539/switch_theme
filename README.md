# Switch Theme

In this Application we can change the theme of application. This application is made using bloc.

## Pages

It has two main pages:
   1. Home page 
   2. Switch Theme Page
   
## Packages Used:
  
  1. Bloc
  2. equatable
  
## Features:

  1. User can change the theme of application.
